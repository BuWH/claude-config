# Claude Configuration Repository

This repository contains configuration files, scripts, and settings for working with Claude AI and related tools.

## Purpose

- Store Claude Code configuration (CLAUDE.md, .claude/)
- Maintain environment variables templates (.env.example)
- Share useful scripts and skill definitions
- Document best practices and workflows

## Structure

- `CLAUDE.md` - Project-specific instructions for Claude Code
- `.env.example` - Template for environment variables
- `.gitignore` - Git ignore rules for Python/Claude projects
- `scripts/` - Useful automation scripts
- `skills/` - Custom Claude skills definitions

## Getting Started

1. Clone this repository
2. Copy `.env.example` to `.env` and fill in your values
3. Install Python dependencies in a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

## Usage

This repository is intended as a centralized location for Claude-related configurations that can be shared across projects and team members.
